STYLING TEXTS

The following content includes the code for styling different text elements on a page (applies to all fonts):

NOTE:
"myGaramond..." fonts were downloaded from Google Fonts and embedded on CSS file via @font-face. The font-weight for each font style was set within @font-face. Therefore when styling the downloaded fonts, you shouldn't need to state the font-weight. For instance:

@CSS:
@font-face {
font-family: "myGaramondLight";
src: url(fonts/Cormorant_Garamond/CormorantGaramond-Light.ttf);
font-weight: 300;
}

p { font-family: myGaramondLight; }

font-size: 50px;
text-align: center;
color: rgb(31, 46, 18);
font-weight: 700;
}

STYLING BOX ELEMENTS

To style box elements you should:

  1. Use the colors shown in the color palette.
  2. Style them to have curved corners
  3. Add padding in order to add some space around the content within each box. This will give them a more tidy and aesthetic appereance.


    4. For example:

    @HTML:
    <div class="topic"> ENTER TOPIC NAME HERE <div>

    @CSS
    .topic {
    border-radius: 20px;
    padding: 15px;
    background-color: rgb(166, 182, 153);
    }
